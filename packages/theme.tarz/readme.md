# Tarz Theme

Plugin Acode yang menambahkan theme Tarz Hot❤️‍🔥🔥

Buka `Pengaturan` -> `Tema` -> `Tema Editor` -> gulir ke bawah dan pilih `Tarz Hot❤️‍🔥🔥`

## Fitur

- glow
- rgb
- syntax
- full warna
- memanjakan mata 🗿✌️
